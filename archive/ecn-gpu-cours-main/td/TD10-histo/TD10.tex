\begin{wtodo}[Histo]{}
Le code fourni dans {\tt histo\_gpu.cu} propose une implémentation sur CPU d'une fonction pour compter le nombre d'occurrences de chaque caractère ASCII dans un texte. Le texte est contenu dans le tableau {\tt h\_str} de {\tt char} qui est déclaré dans le fichier {\tt text.h}. Il y a 128 caractères ASCII et chaque caractère sera placé dans sa propre case pour un total fixe de 128 cases. Les cases de l'histogramme seront des compteurs 32 bits non signés qui ne saturent pas. 

A vous d'implémenter deux versions de ce calcul sur GPU. Une première version sans mémoire partagée et une seconde avec.

Comparer les performances ({\tt gprof} et {\tt nvprof}) avec ou sans l'utilisation de la mémoire partagée par rapport à la version CPU.
\end{wtodo}
